### Installation

##### Install dependencies (git, python)
 - For Debian
    ```
    sudo apt install git python3 -y
    ```
 - For Termux
    ```
    pkg install git python3  -y
    ```
##### Install IcePick
 -  Clone this repository
    ```
    git clone https://github.com/byestumpy/IcePick.git
    ```

 - Enter the directory
    ```
    cd IcePick
    ```

 -  Install all modules
    ```
    pip install -r requirements.txt
    ```

 -  Run the tool
    ```
    python icepick.py
    ```
